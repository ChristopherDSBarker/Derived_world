# PLU OSM Original Pixel Asset Pack

Original pixel-art assets for the uploaded PLU OSM map. The screenshot was used only as broad mood inspiration: warm terracotta roofs, ochre stone, olive accents, and bright blue water. No copyrighted sprites or map layouts were copied.

Includes individual PNGs, packed sheets, Tiled TSX/JSON, preview, legend, and OSM tag-to-asset manifest.

Pixel constraints: no antialiasing, no dithering, limited palette, transparent objects.
